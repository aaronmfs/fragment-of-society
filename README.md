# Fragments of Truth

Simple 2D top-down dungeon game.

Players explore levels.
Fight bosses.
Collect puzzle pieces.
Learn about real social issues.

---

## Goal

Finish all levels.

Collect all puzzle fragments.

Unlock all characters.

Read the notes after each boss.

---

## Gameplay

Top-down pixel dungeon game.

Inspired by:
- Soul Knight
- Super Mario style level progression

Structure:

Level → Enemies → Mid Area → Boss → Note Reward

Levels unlock one by one.

Example:

1-1 → unlocks → 1-2
1-2 → unlocks → 1-3

Replay allowed.
No extra rewards.

---

## Levels

Three themes.

Level 1 – Pollution
Level 2 – Drugs
Level 3 – Corruption

Each theme has three stages.

Example:

1-1 Introduction
1-2 Severity
1-3 Boss

Boss represents the problem in symbolic form.

Example:

Corruption boss → giant crocodile.

---

## Rewards

First clear gives coins.

Normal level: 45 coins
Boss level: 75 coins
Final boss: 80 coins

Coins buy characters.

Characters cost 100 coins.

---

## Puzzle System

Boss defeat gives a note.

Note contains:

- short explanation of a real social issue
- puzzle fragment on the back

Fragments form a larger picture.

All fragments appear in the lobby frame.

Players click fragments to read the note again.

---

## Lobby

Main hub of the game.

Zones:

Center → lounge with NPCs
Left → level portals
Right → character shop
Top → puzzle frame

---

## Characters

Playable characters based on the team members.

Example characters:

Shigairo
Branch Bonker
Oracle
Mobilizer
The Scholar

Each character has:

- unique weapon
- strengths
- weaknesses
- skills

Play.
