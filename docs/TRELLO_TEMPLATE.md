# Trello Card Template

## Template
[System]: verb + what
WHAT: [What needs to be done]
WHY: [Why it matters now]
Checklist
- [ ] item 1
- [ ] item 2
- [ ] ...

---

## Example Card
[Input]: add keyboard and mouse detection
WHAT: Keyboard and Mouse detection
WHY: for player ingame interaction
Checklist
- [ ] mouse
- [ ] keyboard
